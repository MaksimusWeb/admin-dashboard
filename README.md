## Дашборд-админка на Next.js 16+ с PostgreSQL, Tailwind CSS, Vercel host.

Панель управления пользователями.

## Возможности:

- **CRUD - Создание, чтение, обновление и удаление пользователей**
- **Поиск по имени/email, фильтр по ролям**
- **Статистика распределения ролей с визуализацией**
- **Адаптивный дизайн с Tailwind CSS**
- **Автоматическое развертывание на Vercel**

## Стэк:

- **Frontend:** Next.js 16+ (App Router), TypeScript, Tailwind CSS
- **Backend:** Next.js API Routes, PostgreSQL
- **Deployment:** Vercel
- **Database:** Neon (PostgreSQL hosting)

Demo: [Live](https://admin-dashboard-eight-rust-37.vercel.app/)

GitHub: [@MaksimusWeb](https://github.com/MaksimusWeb)
